Exercise 3.6.5. Solve a system of 3 equations.

Consider the system of equations

\begin{equation*} \begin{matrix}4x+8y-4z\amp =4\\3x+8y+5z\amp =-11\\-2x+y+12z\amp =-17. \end{matrix} \end{equation*}

  1. Set up the augmented matrix for this system.

  2. Use row reduction to find the solution.
